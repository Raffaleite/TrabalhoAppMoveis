Siga os passos abaixo para configurar e executar o projeto:

1. **Clone o repositório**

    ```bash
    git clone https://github.com/Raffaleite/TrabalhoAppMoveis.git
    ```

2. **Instale as dependências**

    ```bash
    npm install
    ```

3. **Execute o Expo**

    ```bash
    npx expo start --clear
    ```
